# VRCinvite 使い方ガイド

## 概要

VRCinvite は VRChat のインスタンス作成とユーザー招待をGUIで操作できるツールです。

---

## 起動

アプリケーションを起動すると、前回のセッションが有効な場合は自動的にメイン画面へ遷移します（「セッション確認中...」と表示されます）。セッションが無効な場合はログイン画面が表示されます。

---

## ログイン画面

![](/docs/screenshot2-1.png)

### 手順

1. **ユーザー名** に VRChat のユーザー名を入力します。
2. **パスワード** に VRChat のパスワードを入力します。
3. **ログイン** ボタンをクリックします（またはパスワード欄でEnterキー）。

### 二段階認証

メールまたは認証アプリに届いたコードを入力して **OK** をクリックしてください。

### 注意

- ログインに成功すると、次回起動時からセッションが自動的に再利用されます。
- ユーザー名・セッションCookieは `%APPDATA%\VRCInviteTool\config.json` に保存されます。パスワードは保存されません。

---

## メイン画面

ログイン後、メイン画面に遷移します。画面は3つのセクションで構成されています。

ログイン直後にフレンド一覧とお気に入りワールドがバックグラウンドで自動取得されます。右上の **更新** ボタンで再取得できます。

---

### セクション1: インスタンスを作成

![](/docs/screenshot2-2.png)

ワールドにインスタンスを作成します。

#### ワールドの指定方法

ワールドは2つの方法で指定できます。

1. **ワールド検索** — 「ワールド検索 (名前 or ID)」フィールドにワールド名またはIDの一部を入力すると、お気に入りワールドから候補が表示されます。候補をクリックすると「ワールドID」が自動入力されます。フォーカスするだけで一覧も表示されます。
2. **直接入力** — 「ワールドID」フィールドにワールドIDを直接入力します。

#### 各フィールド

| フィールド | 説明 |
|-----------|------|
| ワールド検索 (名前 or ID) | お気に入りワールドから名前またはIDで検索 |
| ワールドID | インスタンスを作成するワールドのID（例: `wrld_xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`） |
| インスタンスタイプ | インスタンスの種類（`Public` / `Friends` / `Friends+` / `Invite` / `Invite+`） |
| リージョン | サーバーリージョン（`jp` / `us` / `use` / `eu`） |

#### ワールド情報の自動表示

「ワールドID」を入力すると、自動でワールド情報（名前・サムネイル）が検索され、右側に表示されます。

#### 手順

1. ワールド検索または「ワールドID」でワールドを指定します。
2. **インスタンスタイプ** でインスタンスの種類を選択します。
3. **リージョン** でリージョンを選択します。
4. **インスタンスを作成** ボタンをクリックします。
5. 作成が完了すると、**インスタンス場所** フィールドにインスタンスIDが表示されます。
6. コピーアイコンをクリックするとクリップボードにコピーできます。

---

### セクション2: 招待

![](/docs/screenshot2-3.png)

指定したユーザーを指定したインスタンスに招待します。最大20人まで一括招待が可能です。

#### ユーザーの指定方法

1. **フレンド検索** — 「フレンド検索 (名前 or ID)」フィールドにフレンド名またはIDの一部を入力すると候補が表示されます。候補をクリックすると選択チップとして追加されます。フォーカスするだけで一覧も表示されます。
2. **直接入力** — フレンドを選択していない状態で、フレンド検索フィールドにユーザーIDを直接入力して招待することもできます。

#### 各フィールド

| フィールド | 説明 |
|-----------|------|
| フレンド検索 (名前 or ID) | フレンド一覧から名前またはIDで検索・選択（複数選択可、最大20人） |
| インスタンスID | 招待先のインスタンスID（インスタンス作成後は自動入力されます） |

#### 選択状態の表示

- フレンドを選択すると「選択中: X / 20」のカウンターが表示されます。
- 選択したフレンドはチップとして表示され、×ボタンで個別に削除できます。
- 同じフレンドを重複して選択することはできません。

#### ユーザー情報の自動表示

最初に選択したフレンドのユーザー情報（表示名・アイコン）が右側に表示されます。

#### 手順

1. フレンド検索からユーザーを選択します（複数可）。
2. **インスタンスID** に招待先のインスタンスIDを入力します（インスタンス作成後は自動で入力されます）。
3. **招待する** ボタンをクリックします。

#### 自分に招待を送る

**自分に招待を送る** ボタンをクリックすると、ログイン中のアカウント自身にインバイトを送信します。

#### 複数人招待時の動作

- 各招待の間に1秒の遅延が設けられています。
- APIエラー（403等）が発生した場合はエラーメッセージを表示し、次の人への招待を続行します。
- ネットワークエラー等の致命的なエラーの場合は招待ループ全体が中断されます。

#### 注意

- 招待できるのはフレンドのユーザーのみです。フレンドでない場合はエラーが表示されます。

---

### セクション3: ログ

![](/docs/screenshot2-4.png)

アプリケーションの動作ログが表示されます。

- ワールド名・ユーザー名の取得結果、招待の送信状況、エラーがリアルタイムで表示されます。
- **クリア** ボタンでログをリセットできます。

---

## ログアウト

画面右上の **ログアウト** ボタンをクリックするとセッションが削除され、ログイン画面に戻ります。

---

## エラーについて

| エラーメッセージ | 原因 |
|----------------|------|
| ユーザー名とパスワードを入力してください | 未入力のフィールドがあります |
| ログインに失敗しました | 認証情報が正しくありません |
| セッションが切れました。再度ログインしてください。 | セッションが期限切れです。再ログインしてください |
| データ取得エラー | フレンド一覧またはワールド情報の取得に失敗しました |
| ワールドが見つかりません | ワールドIDが存在しないか、形式が誤っています |
| ユーザーが見つかりません | ユーザーIDが存在しないか、形式が誤っています |
| ログインしていません | セッションが失われています。再ログインしてください |
| フレンドではないため招待できません | 招待先のユーザーとフレンド関係がありません |
| 選択上限(20人)に達しています | フレンド選択が上限の20人に達しています |
| インスタンスIDを入力してください。 | 招待先のインスタンスIDが空です |
| ユーザーを選択または入力してください。 | 招待対象が選択・入力されていません |
| location形式が不正です | インスタンスIDの形式が正しくありません |
